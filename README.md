
Purpose of the Code

This project transforms the Y-P Fibonacci LFSR-based stream cipher into an equivalent stream cipher based on the Espresso Galois NFSR F using the modified Y-P Fibonacci-Type II Galois transformation algorithm.
The primary objective of the implementation is to compare the output sequences generated by the original Fibonacci configuration and the transformed Galois configuration, ensuring functional equivalence under different compensation strategies.
Two types of compensation techniques are considered during the transformation process:
	Simple Compensation
	Iterative Compensation (performed in descending order)
The code evaluates and compares the output sequences for four different combinations of these compensation methods to study their correctness and behavioral equivalence.
The combinations examined are:

	Output function: Iterative, Initial state: Simple
  
	Output function: Simple, Initial state: Iterative
  
	Output function: Iterative, Initial state: Iterative
  
	Output function: Simple, Initial state: Simple

In the github repository, there are four sage files support the above combinations:

Fibo2GaloisF_IterativeOutputFn_SimpleInitState.sage

Fibo2GaloisF_SimpleOutputFn_IterativeInitState.sage

Fibo2GaloisF_SimpleOutputFn_SimpleInitState.sage

Fibo2GaloisF_IterativeOutputFn_SimpleInitState.sage

All programs run on SageMath version 10.7, Release Date: 2025-08-09.
